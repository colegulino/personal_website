Introduction

This webpage is designed to contain all of the information for my final project for the course 16-745 Dynamic Optimization at Carnegie Mellon University

The project that I will be completing will be designed around Partially Observable Markov Decision Processes in order to learn a value function representing a policy for acting in urban environment with pedestrians. Specifically I would like to learn how to handle the uncertainty in a pedestrian's actions.

I am also interested in using parallel computation for efficient policy optimization as suggested in the project suggestion page

Project Description

For this project, I plan to use methods similiar to [8], [9], and [10] in order to form a policy for planning with an autonomous vehicle that must take into account the actions of the pedestrians around it.

I plan to utilize methods in [1], [2], [3], and [7] in order to solve for the policy that optimizes the decisions of the car as it depends on the pedestrians.

I plan on either running the alogithms from images using similiar data sets to [8], [9], or [10] or by using the openrave simulation environment.

I plan to learn a policy and demonstrate in simulation or through test images the actions of an autonomous car in an urban environment with pedestrains.

Finally, I am considering using parellel computation to efficiently compute the notoriously computationally intensive POMDPs as suggested in the project suggestion page
