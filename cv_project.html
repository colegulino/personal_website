Introduction

This the webpage for my Computer Vision Project, which I am working on for my Spring 2016 course, 16-720 Computer Vision at Carnegie Mellon University.

The project is based on the idea of recognizing pedestrians from geometries using Semantic Structure from Motion. I believe that this is a worthwhile task especially in self-driving car applications. Recognition using Structure from Motion could improve recognition rates of pedestrians in low-light settings

I plan to use the Caltech Pedestrian Detection Dataset

Project Description

A major challenge for autonomous vehicles is recognition of dynamic obstacles such as pedestrians in cluttered urban environments. This problem can be exacerbated by low-light conditions such as driving in the rain or at night. Normal detection methods may not be accurate in these low-light settings. Another major issue with single image classification methods such as bag of words, is that the do not provide valuable information about the pedestrians such as orientation, and distance in the 3D scene. This information is very important for understanding the intentions and probability of collisions for dynamic obstacles. I propose that by using the Semantic Structure from Motion Algorithm [1] along with techniques for improving Structure from Motion in low-light settings [2], I will be able to get accurate pedestrian classification and tracking in low light settings.

I will be implementing the algorithms myself in C++ utilizing OpenCV. I will not use an out of the box method for the Structure from Motion algorithm. I plan to specifically implement the Semantic Structure from Motion Algorithm [1] while attempting to improve performance in low-light settings. I plan to initially use the Caltech Pedestrian Detection Benchmark to test my performance along with the datasets provided [3]. [4], [5], and [6] show other relevant papers.
